L’adresse de notre site Web est : https://multibricolage18.fr. Le site est amené à collecter et à traiter des informations dont certaines sont qualifiées de « données personnelles ». Nous respectons votre vie privée et n’utilisons vos données que pour une finalité précise.

Utilisation des données personnelles collectées

Sur le site www.multibricolage18.fr, il y a un seul type de données.

Les données transmises directement :

Ces données sont celles que vous nous transmettez directement, via les deux formulaires de contact ou de demande de devis. Sont obligatoires dans les formulaires de contact, les champs : « Nom et prénom », « email » et « téléphone ».

Les données que vous nous transmettez directement sont utilisées dans le but de vous re-contacter et/ou dans le cadre de la demande que vous nous faites.

Les données seront sauvegardées durant une durée maximale de 3 ans.
